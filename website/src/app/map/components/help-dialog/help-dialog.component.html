Editing Controls

Polygons:

  • Left Click: Adds new polygon point
  • Double Click: Finishes polygon drawing
  • Esc Key: Finishes drawing without adding point
  • Ctrl + Right Click: Deletes polygon point
  • Right Click (on finished polygon): Opens context menu

Building Points:

  • Left Click: Selects the building
  • Left Click + Drag: Selects the building and changes it's position
  • Enter Key: Locks the selected building and moves to next building
  • Shift + Enter Key: Unlocks the selected building
  • Space Key: Locks the selected building
  • Ctrl + Left Click: Adds new building on the map
  • Ctrl + Right Click: Deletes the building
  • Left Arrow Key: Selects previous point
  • Right Arrow Key: Selects next point

Network:

  • Left Click: Adds new point
  • Ctrl + Left Click (on existing network point): Creates a new point and links it to existing segment
  • Double Click: Completes current segment
  • Esc Key: Finishes drawing without adding point
  • Right Click (on existing network point): Open context menu
  • Ctrl + Right Click (on existing network point): Deletes the point
